# QNX_Learning
This Repo is for learning Realtime Programming for the QNX Operating System Course from QNX provided course on E-Learning

you can find the course at https://learning.qnx.com/qnx/courses/20207

The course is divided into 10 modules that are bookended by Introduction and Conclusion lessons. Each of the 10 modules have various lessons that cover the topical subjects for the course.

## Course Modules
0. Prerequisites and Exercises for Download
1. QNX Architecture
2. Momentics Development Basics
3. Introduction to Security Policies
4. Processes, Threads & Synchronization
5. QNX Interprocess Communication
6. Comparing QNX IPC Methods
7. Introduction to Hardware Programming
8. Timers, Clocks and Timeouts
9. Build a QNX Neutrino Boot/OS Image
10. Introduction to Resource Managers
